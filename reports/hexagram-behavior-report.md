## Methodology

1. Run wiring through MMCA for N generations
2. Sample random (t, x) positions from run history
3. Extract 36-bit exotype context at each sample:
   - LEFT (8 bits): predecessor sigil
   - EGO (8 bits): self sigil
   - RIGHT (8 bits): successor sigil
   - NEXT (8 bits): next-generation sigil
   - PHENOTYPE (4 bits): phenotype family
4. Arrange into 6x6 matrix and compute eigenvalues
5. Signs of 6 eigenvalues → hexagram lines
6. Map to King Wen hexagram number

See `futon5.hexagram.lift` and `futon5.mmca.exotype` for implementation.
